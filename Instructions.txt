ENG

-First we go to the Discord Developer Portal, create an app with the name of Retro MMO and copy the APPLICATION ID
-Then we are going to Rich Presence and add an image as a asset (the one in the img folder) and named it logo, js reads that name so is needed to be called like that
-Create or modify the .env file in the same folder along the app.js and type inside: "applicationID=[yourid]" without the squarebrackets and quotation marks
-Your applicationID should look something like this: applicationID=123456789012345678
-Enter the Config.json file and change the username
-Save changes and you are done!
